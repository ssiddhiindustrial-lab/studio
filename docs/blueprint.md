# **App Name**: Siddhi Build

## Core Features:

- Industrial Portfolio Engine: A dynamic project showcase driven by Firestore, allowing visitors to filter work by category (Ongoing, Completed, Warehousing) and view detailed project specifications.
- Project Cost Intelligence Tool: An AI construction advisor tool that analyzes raw client requirements to generate high-level scope suggestions and connects them with historical project data.
- Industrial Inquiry Hub: A high-conversion lead generation form that captures client project types and syncs submissions directly to Firestore for real-time tracking.
- Precision Timeline: An interactive vertical timeline UI illustrating the company's evolution from 1997 site engineering to the founding of Siddhi Industrial Services.
- Compliance & Safety Monitor: A dedicated section showcasing HSE protocols, worker welfare programs, and ISO compliance standards with visual KPI indicators.
- Visual Asset Gallery: A masonry layout gallery integrated with Firebase Storage for fast loading of site activity images and high-resolution industrial photography.
- Automated SEO & Analytics: Server-side rendering (SSR) of project detail pages to ensure each high-value construction contract is indexable by search engines.

## Style Guidelines:

- Primary Color: Professional Deep Navy (#243B55) representing reliability and engineering authority.
- Accent Color: Safety Orange (#FF8C00) for high-visibility call-to-action buttons and critical interface elements, contrasting against a Clean Paper White (#F7F9FC) background.
- Headline font: 'Space Grotesk' (sans-serif) for a modern, precise, and tech-driven industrial look. Body font: 'Inter' (sans-serif) for objective readability.
- A mobile-first, grid-based layout with significant negative space to emphasize structural clarity and architectural scale.
- Thin-line monolinear icons for service categories, emphasizing technical drawing aesthetics over generic visuals.
- Industrial-grade smooth scrolling and subtle reveal transitions on hover to signify stability and precision engineering.
